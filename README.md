# Crescent Tuition Center — Business Website

## Overview
Designed and deployed a responsive single-page website for a local tuition 
center with a dynamic enrollment form, conditional subject-selection logic, 
and automated email notifications via the Formspree API.

## Live Site
🔗 https://crescenttuitioncenter.netlify.app

## Key Features
- Dynamic enrollment form with conditional subject-selection logic
- Automated email notifications via Formspree API
- Course filtering and form validation
- WhatsApp Community integration
- Fully responsive single-page design

## Tools Used
- HTML
- CSS
- JavaScript
- Formspree API
- Netlify (Hosting)

## Impact
Replaced manual paperwork with a self-service digital enrollment process 
for the tuition center.
